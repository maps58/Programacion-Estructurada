# Programacion-Estructurada
Codigos para mis estudiantes del Tecnologico de Oaxaca que serán explicados en la materia de PROGRAMACION ESTRUCTURADA, en especifico sob códigos en lenguaje C.

La programación estructurada es un paradigma de desarrollo de software centrado en mejorar la claridad, calidad y tiempo de desarrollo, utilizando tres estructuras de control fundamentales: secuencia, selección (if/else) e iteración (bucles). Evita saltos desordenados (código espagueti) mediante subrutinas o funciones, facilitando el mantenimiento y depuración del código. 
Características y Fundamentos:

    Tres estructuras lógicas: Secuencia (pasos ordenados), selección (toma de decisiones) e iteración (repetición).
    Diseño top-down: Divide problemas complejos en subproblemas más pequeños y manejables.
    Eliminación del 'goto': Evita saltos incondicionales de flujo que dificultan la lectura del código.
    Código legible: Utiliza sangrías (indentación) para organizar bloques de código.

    Ventajas principales:

    Mayor legibilidad: Los programas son más fáciles de entender y mantener.
    Depuración sencilla: Facilita la localización y corrección de errores.
    Reutilización: Permite dividir el código en funciones o módulos reutilizables. 

Lenguajes comunes: Lenguajes como C, Pascal, Python y Java permiten la implementación de esta metodología. 
Teorema de la estructura: Propuesto por Böhm y Jacopini en 1966, demuestra que cualquier programa puede ser escrito utilizando únicamente las tres estructuras mencionadas, lo que garantiza que todo programa propio se puede estructura.
